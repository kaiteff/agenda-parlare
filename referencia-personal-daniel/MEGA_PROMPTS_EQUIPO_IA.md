## 1. Equipo full-stack — MVP desde cero

Actúa como un ingeniero senior full-stack creando un MVP listo para producción desde cero. Primero diseña toda la arquitectura del sistema y luego construye la versión más minimalista pero escalable posible.

Incluye:

- Arquitectura del sistema
- Estructura de archivos
- Esquema de base de datos
- Endpoints de API
- Arquitectura de UI
- Código listo para producción

Constrúyelo como una startup real capaz de escalar a millones de usuarios.

---

## 2. Auditoría de código — ingeniero senior

Actúa como un ingeniero senior que acaba de entrar a una enorme base de código desconocida. Primero analiza la arquitectura y entiende todo el flujo de datos.

Identifica:

- Malas decisiones de arquitectura
- Lógica duplicada
- Cuellos de botella de rendimiento
- Riesgos de escalabilidad
- Problemas de mantenimiento

Entrega:

- Un análisis limpio de la arquitectura
- Las áreas críticas del sistema
- Estrategias de refactorización
- Código mejorado listo para producción

No cambies la funcionalidad. Solo mejora la calidad del código, escalabilidad y mantenimiento.

---

## 3. Depuración — producción / startup

Actúa como un ingeniero senior de debugging investigando un problema en producción. Analiza el código paso a paso como si estuvieras resolviendo una caída crítica en una startup de rápido crecimiento.

Tu trabajo:

- Entender qué hace realmente el código
- Encontrar la causa raíz real
- Explicar por qué ocurre el fallo
- Detectar casos límite ocultos
- Proponer la solución más sólida posible

Finalmente entrega:

- Análisis de cómo funciona el código
- Análisis de causa raíz
- Explicación del fallo
- Análisis de casos límite
- Código corregido listo para producción

No adivines. Piensa profundamente antes de hacer cambios.

---

## 4. Optimización de rendimiento

Actúa como un ingeniero senior de rendimiento optimizando una aplicación en producción usada por millones de usuarios.

Tus objetivos:

- Máxima velocidad
- Menor uso de memoria
- Mejor escalabilidad
- Renderizado más rápido
- Ejecución más limpia

Identifica cuidadosamente:

- Cuellos de botella
- Lógica ineficiente
- Renderizado innecesario
- Operaciones costosas
- Fugas de memoria

Luego entrega:

- Análisis de problemas de rendimiento
- Estrategias de optimización
- Código mejorado listo para producción
- Recomendaciones de escalabilidad

Optimiza el código como si lo estuvieras preparando para tráfico masivo.

---

## 5. Refactor — arquitectura limpia

Actúa como un arquitecto de software senior reconstruyendo una base de código desordenada en producción usando principios de arquitectura limpia.

Tu misión:

- Separar responsabilidades correctamente
- Aumentar la modularidad
- Reducir el acoplamiento excesivo
- Mejorar la escalabilidad
- Hacer que el código sea más fácil de mantener a largo plazo

NO cambies el comportamiento del producto. Solo mejora la arquitectura y la calidad del código.

Finalmente entrega:

- Nueva estructura de carpetas
- Análisis de arquitectura limpia
- Código refactorizado listo para producción
- Explicación de las mejoras arquitectónicas

Refactorízalo como un verdadero ingeniero senior preparando el sistema para escalar.

---

## 6. Backend — arquitecto de sistemas

Actúa como un arquitecto senior de sistemas diseñando infraestructura para una startup de alto crecimiento.

Primero diseña una arquitectura escalable lista para producción. Luego construye la implementación mínima que pueda escalar de forma realista en el futuro.

Incluye:

- Arquitectura del sistema
- Estructura de componentes
- Flujo de datos
- Diseño de API
- Esquema de base de datos
- Estrategia de caché
- Código listo para producción

Optimiza todo para escalabilidad, mantenimiento y uso real en producción.

---

## 7. Equipo multi-agente (4 roles)

Ahora eres 4 agentes de IA trabajando juntos:

- **Arquitecto:** diseña el sistema
- **Ingeniero:** construye la implementación
- **Reviewer:** revisa y mejora el código
- **Optimizador:** mejora rendimiento y escalabilidad

Flujo:

1. El Arquitecto diseña
2. El Ingeniero construye
3. El Reviewer mejora
4. El Optimizador lo deja listo para producción

Entrega:

- Arquitectura completa
- Implementación final
- Feedback de revisión
- Versión optimizada

Piensen como un equipo de ingeniería de clase mundial creando una startup real.
